# Threads 20投稿ジェネレーター（MVP）

Threadsで毎日20投稿の下書きを作るための、初心者向けシンプルWebアプリです。

## できること

- 今日のテーマを入力
- 今日の狙いを選択
- 口調を選択
- ボタン1つで20投稿を自動生成
- 投稿ごとに「番号」と「カテゴリラベル」を表示
- 個別コピー / 20件まとめてコピー

## ローカル起動手順（初心者向け）

1. ターミナルでこのフォルダに移動します。
2. 次のコマンドを実行してローカルサーバーを起動します。

```bash
python3 -m http.server 8000
```

3. ブラウザで `http://localhost:8000` を開きます。
4. 画面で以下を入力・選択します。
   - 今日のテーマ
   - 今日の狙い
   - 口調
5. `20投稿を生成する` を押します。
6. 必要に応じて
   - 各投稿の `コピー` ボタンで個別コピー
   - `20投稿をまとめてコピー` ボタンで一括コピー

## docs との連携

このMVPは、以下のファイルを読み込んで生成に利用します。

- `docs/sample-posts.md` の見出し（`##`） → カテゴリラベル
- `docs/ng-words.md` の箇条書き（`-` / `*`） → NGワード置換

※ 現在のdocsは見出し中心のため、まずはテンプレートベースで生成します。
