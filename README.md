# Threads 20ツリー投稿ジェネレーター（MVP）

るナース専用の、2投稿1セットのThreadsツリーを20セット作るWebアプリです。

## できること

- 今日のテーマ一覧（1行1テーマ）を入力
- ボタン1つで20セットを自動生成
- 各セットに `No. / テーマ / 型 / 投稿1 / 投稿2` を表示
- 投稿1・投稿2・ツリー全体の個別コピー
- 20セット全体の一括コピー
- コピー成功時に `コピーしました` を表示（2秒で元に戻る）
- コピー失敗時に案内メッセージを表示

## ローカル起動手順

1. ターミナルでこのフォルダに移動します。
2. 次のコマンドを実行してローカルサーバーを起動します。

```bash
python3 -m http.server 8000
```

3. ブラウザで `http://localhost:8000` を開きます。
4. 今日のテーマ一覧を1行ずつ入力します。
5. `20セットを生成する` を押します。

## 生成ルール（固定）

- 発信者：るナース
- 対象：note初心者 / 副業初心者 / 30〜40代 / 看護師 / 在宅ワークに興味がある人
- 投稿形式：2投稿1セット
- 投稿1：フック → 共感 → 失敗・気づき → 続きが気になる終わり方
- 投稿2：答え → 理由 → 経験 → 今日できる小さな行動 → やさしい締め
- 一人称：`わたし` 固定
